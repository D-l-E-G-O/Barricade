# Barricade

A strategic wall-and-pawn racing game. Build mazes, trap your opponent, and race to the finish line. 
A modern, full-stack TypeScript implementation of the classic board game Quoridor.

## Architecture
- **Backend & Game Logic**: TypeScript (Node.js)
- **Web Interface**: TypeScript (Web Framework / HTML / CSS)

## Development Milestones
1. Core game logic (Board representation, pathfinding, move validation).
2. Local interactive web interface (Frontend only).
3. Multiplayer Server Integration (Node.js + WebSockets).
